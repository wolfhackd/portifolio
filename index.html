Sobre Mim

Sou Mauro, e estou apenas começando minha jornada como estudante de Análise e Desenvolvimento de Sistemas. Este site é um espaço onde compartilho meu portfólio em constante crescimento, repleto de ilustrações e designs que refletem minha paixão pela arte. Cada peça conta uma história única, moldada por minha dedicação à excelência e à inovação. Convido você a explorar este universo de cores e formas, onde a criatividade não conhece limites. Obrigado por se juntar a mim nesta emocionante jornada. Vamos inspirar e ser inspirados enquanto mergulhamos juntos no maravilhoso mundo da arte, do design e da tecnologia.

GitHub

Plataforma de hospedagem de código, utilizada para colaboração, controle de versão e compartilhamento de projetos de software.

JavaScript

Linguagem de programação amplamente utilizada para desenvolver aplicativos web interativos e dinâmicos, bem como aplicativos de servidor e dispositivos móveis.

HTML

Linguagem de marcação fundamental para a criação de páginas web, definindo a estrutura e o conteúdo dos documentos exibidos nos navegadores da web.

CSS

(Cascading Style Sheets) é uma linguagem de estilo utilizada para controlar a apresentação visual de páginas web, incluindo layout, cores e fontes.
